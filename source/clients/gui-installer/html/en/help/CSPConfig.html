SSL certificate configuration

Screen is displayed only if CSP or JMX features were enabled. SSL certificate will be generated as part of qmaster installation. This certificate will be than used throughout the Grid Engine.

Following attributes will be displayed in the SSL certificate:

  • Country code - 2 character country code. By default DE.

  • State – State. By default GERMANY.

  • Location – Location. By default Building.

  • Organization – Organization. By default Organisation.

  • Organization unit - Organization unit. By default Organisation_unit.

  • Email address - Email address. By default name@yourdomain.com.
