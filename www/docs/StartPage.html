My Start Page: sections
Pending Actions

Your pending activities on this site are tracked here: projects you have proposed awaiting approval from site administrators, your requests to join other hosted projects, and project documents you have agreed to review.

If you already have an approved development project on this site, this section features additional pending project administrative items awaiting your approval as the Project Owner, including:

  • Proposed new project members
  • Proposed new or changed roles for project members
  • Proposed news items
  • Proposed files and documents
My Projects

This list captures all the hosted development projects which you have joined and have the role of observer, code developer, or content developer. If you have started your own project, this also appears along with a link to a project administration page. More about project roles.

Hosted Project Categories

All projects currently hosted on this site are listed by category. Projects are grouped into categories according to the development nature or type of application. Thus, any projects dealing with games are associated together. As a result, you can quickly discover what sort of projects are available in your areas of interest and expertise. Category names in this section link to category home pages, where projects are listed by individual name with short descriptions. More about project categories
